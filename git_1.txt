УР - удалённый репозиторий
ЛР - локальный репозиторий


--- УР ---
hello_world.py
funcs.py



--- ЛР ---
hello_world.py
funcs.py

шаг 0: git clone - скопировал все файлы и историю из УР в каталог локальной файловой системы и создал ЛР в этом каталоге

шаг 1: внёс изменения в файл hello_world.py

шаг 2: git add - запланировал изменения для следующего коммита

шаг 3: git commit -m - зафиксировал все запланированные на шаге 2 изменения, создал коммит

шаг 4: git push - отправил коммиты из ЛР в УР, синзронизировал ветки
